# Bin2Dec App

A simple browser-based utility that converts binary strings into their corresponding decimal integers. Built as part of the tier-1 Beginner Projects collection.

## Features

- **Strict Validation:** Real-time feedback if a user enters digits other than `0` or `1`.
- **Length Constraint:** Accepts up to 8 binary digits (1 Byte limits) per the original spec.
- **Pure Math Implementation:** Calculates the decimal output using basic mathematical positioning rules instead of high-level native typecasts.
- **Responsive Web UI:** Clean, dark-themed UI that works well on mobile and desktop viewports.

## How to Run

1. Clone or download this repository.
2. Open the `index.html` file directly in any modern web browser.
3. Type your binary number and hit **Convert** (or press `Enter`).
