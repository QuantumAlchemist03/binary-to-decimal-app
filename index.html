How Binary-to-Decimal Works

Binary is a **base-2** numbering system where each column represents a power of 2, starting from the rightmost bit ($2^0 = 1$).

To convert any binary number to a decimal, multiply each binary digit ($0$ or $1$) by its corresponding power of 2, then sum the results:

Value = (dn × 2n) + ... + (d1 × 21) + (d0 × 20)

For example, the binary string 1011 has bits active at positions 3, 1, and 0:

(1 × 8) + (0 × 4) + (1 × 2) + (1 × 1) = 8 + 0 + 2 + 1 = 11
